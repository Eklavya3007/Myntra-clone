# 🛍️ Myntra Clone

This is a front-end clone of the Myntra website built using HTML, CSS, and JavaScript. The project replicates the core design and basic functionalities of an e-commerce platform, focusing on user interface and responsiveness.

## 🚀 Features
- Responsive homepage design
- Product listing page
- Navigation bar with categories
- Add to cart functionality (basic)
- Interactive UI using JavaScript
- Clean and modern design inspired by Myntra

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
